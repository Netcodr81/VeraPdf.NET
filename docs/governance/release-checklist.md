# Release Governance Checklist

- [ ] Build and test pass on Windows, Linux, and macOS CI matrix.
- [ ] SBOM dependency manifest generated and archived.
- [ ] NuGet package metadata validated (license, repository, symbols).
- [ ] Runtime archive checksums updated and verified.
- [ ] Security review completed for process execution and temp-file handling.
- [ ] Signing/attestation notes reviewed for current release.
